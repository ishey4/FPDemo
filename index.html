Purpose

This demo will show a few things
  • How functional vs OO works
  • How expensive renders can get
  • How rendering every little change as opposed to all changes effects processing

Usage / Setup

This page generates counters on the page, and update them.
  • Counters - how many counters to render
  • Render Type - will each counter render itself, or will render happen on the next event loop
  • Duration - how long the counters will count for
  • Wait - how long each counter will wait before next count
  • Go - Spawns counters.
